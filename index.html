Basics for Beginners

Before starting python course you should be aware of some following basic terms that will also help to start programming in python language.


Constants:

The values that are not expected to change during the execution of program.

Example:

34 , 20.67 etc.


Variables:

The values that can be changed during the execution of program. It holds the data inside it that might be any number, quantity or any characteristics.

Example:

Number = 78 , Name = “Hashir”, City = “Attock”
Here Number, Name and City are variables


Comments:

The comments are the statements that are written for humans so that they can understand the code easily. Comments are not executed by compilers or interpreters. The symbol indicating comments varies from language to language. In python, for writing comments “#” symbol is used.

Example:

Num1 = 6 # here Num1 is the first number
Num2 = 9 # here Num2 is the second number


Declaring Variables in Python:

We do not have to declare a variable with any specific data type in python. It automatically determines the data type of variable as soon as the value is assigned to it.

Example:

Age = 42
Salary = 58000


How to print in Python:

The printing in python is the most easier in comparison to any other programming language. To print in python you must need to use the built-in function print().

Example:

print(“I am Alexa”)
Output: I am Alexa
